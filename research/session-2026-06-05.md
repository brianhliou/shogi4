## 1. What's built + how to reproduce it

**Engine / solver (`solver/`, Rust crate `shogi4`).** `cargo test --release` —
10 tests, ~9 s, green — reproduces every core claim: perft from the start, the
cross-language move-gen golden diff, all three solvers agreeing, the dense-rank
bijection + its domain == 2× the enumerator, un-move-gen == reverse edges +
determinism, LR-symmetry value-invariance, and the audit certifying clean +
catching corruption.

**Distributed-design + scaling checks (`solver/src/bin/`):**
- `sharded_check` — sharded/BSP/shuffle solve == oracle for every shard count (confluent).
- `failrec_check` — crash + deterministic replay; result byte-identical to crash-free.
- `audit_demo` — verification is ~½ the solve cost and catches injected corruption.
- `scale [cap]` — the in-RAM calibration ladder (largest: 51.5M positions).
- `rank_check`, `push_check`, `pred_check`, `flat_check` — the per-component proofs.

**Reference oracle (`engine/`):** `shogi4.py` (faithful port of the decompiled
app), `gen_golden.py` / `golden.txt`, `gen_subgames.py` / `subgame_expected.txt`.

**Numbers (see `findings.md`):** rank domain N = 4.10×10¹⁴ (= 2× the exact
arrangement count); reachable ~3×10¹³; W/L/D flat array ~50–100 TB (exact-2× LR
fold); compute ~60–190 core-years; ~$5–15k bare-metal. Measured ns/edge ~330–490
(rank-based). Largest partial: `{2K+carp+fox+raccoon}` = 51,461,568 positions
(W 80.3 / L 18.9 / D 0.8 %).

## 4. Relationship to `../micro-shogi`

Micro Shogi (the bigger rung) is **paused**; its own session recommends solving 4×4
first as the de-risk, and the production architecture it wants — combinatorial rank,
un-move-gen, SCC/BSP streaming, deterministic checkpoint-recovery — is exactly what
Shogi4 has already built and validated. So **Shogi4's solver is the production solver
Micro inherits.** Cost cross-check: Micro's 167 ns is a HashMap floor; our rank-index
pays a combinatorial per-edge cost (~330–490 ns), the price of the only index that
scales — both repos' working footprints are the arrangement domain, not reachable.
